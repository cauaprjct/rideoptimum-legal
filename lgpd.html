Informações sobre a LGPD

Última atualização: 8 de maio de 2026

Em conformidade com a Lei Geral de Proteção de Dados (Lei 13.709/2018), apresentamos as informações sobre tratamento de dados pessoais no Ride Optimum.

1. Controlador de dados

Nome: Cauã Alves
Email: suporte@rideoptimum.com
Atividade: Desenvolvimento de aplicativo auxiliar para motoristas de app.

2. Dados pessoais tratados

O Ride Optimum coleta dados mínimos:

  • Dados de diagnóstico anônimos (textos extraídos de telas de apps de corrida)
  • Dados de uso agregados (Firebase Analytics)
  • Logs de erro (Firebase Crashlytics)

Não coletamos: nome, CPF, endereço, dados bancários, localização GPS contínua, ou qualquer dado pessoal do passageiro.

3. Base legal

O tratamento de dados é baseado no consentimento (art. 7º, I da LGPD) para uso do Serviço de Acessibilidade e no legítimo interesse (art. 7º, IX) para melhoria do app via analytics/crashlytics.
